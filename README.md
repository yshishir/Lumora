# Lumora

Lumora is a modern, minimal blog app built for distraction-free reading and writing.  
It focuses on clean design, simple publishing, and a smooth reading experience.

## Features

- Modern and minimal UI
- Distraction-free blog reading experience
- User authentication with login, register, and logout
- Create and publish blogs
- Rich text blog content support
- Blog cards with title, description, thumbnail, author, and date
- Single blog detail page
- Responsive layout
- Clean navigation with protected write flow
- Image upload support for blog thumbnails
- REST API based backend integration

## Tech Stack

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS
- Shadcn UI
- Tiptap Editor
- React Icons

### Backend

- Node.js
- Express.js
- Prisma ORM
- PostgreSQL

- bcrypt

### Services

- Neon PostgreSQL
- Cloudinary
